## Development

To start the development server, follow these steps:

1. Navigate to the project directory:
    ```sh
    cd "D:\Simple webpage\vite-project"
    ```

2. Reinstall dependencies:
    ```sh
    npm install
    ```

3. Start the development server:
    ```sh
    npm run dev
    ```
